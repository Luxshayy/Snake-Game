# 🐍 Snake Game

A modern **browser-based Snake Game** built with **HTML5 Canvas and JavaScript**.
The game recreates the classic arcade experience with smooth visuals, mobile support, and dynamic gameplay.

---

## 🎮 Play the Game

👉 **[Play Snake Game](https://luxshayy.github.io/Snake-Game/)**

---

## ✨ Features

* 🎮 **Start Screen** – Press Enter or Tap to begin
* 🐍 **Classic Snake Gameplay**
* 👀 **Snake Head with Animated Eyes**
* 🍎 **Animated Pulsing Apple**
* 📈 **Live Score System**
* 🏆 **High Score Saved in Browser (LocalStorage)**
* ⚡ **Dynamic Difficulty** – Snake speed increases with score
* 💀 **Game Over Screen**
* 🔄 **Quick Restart System**
* ✨ **Neon Glow Snake Effect**
* 📱 **Mobile Support**

  * Tap to start
  * Swipe to control movement

---

## 🎯 Controls

### 💻 Desktop

| Key            | Action                  |
| -------------- | ----------------------- |
| **Enter**      | Start Game              |
| **Arrow Keys** | Move Snake              |
| **Space**      | Restart After Game Over |

### 📱 Mobile

| Gesture         | Action               |
| --------------- | -------------------- |
| **Tap Screen**  | Start / Restart Game |
| **Swipe Up**    | Move Up              |
| **Swipe Down**  | Move Down            |
| **Swipe Left**  | Move Left            |
| **Swipe Right** | Move Right           |
